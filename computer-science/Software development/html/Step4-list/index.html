# 列表的作用

列表是指将具有相似特征或者具有先后顺序的几行文字进行对齐排列
[所有的列表都由列表类型和列表项组成]

# 有序列表 <ol>

<ol> 元素编写有序列表，用于列出页面上有特定次序的一些项目

# 无序列表 <ul>

<ul> 元素表示无序列表，用于列出页面上没有特定次序的一些项目

# 列表嵌套

将列表元素嵌套使用，可以创作多层列表
