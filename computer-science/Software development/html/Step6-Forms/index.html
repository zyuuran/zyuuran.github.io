# 表单概述

# 表单的作用

表单用于显示、收集信息，并提交信息到服务器
表单有两个基本部分
1、实现数据交互可见的页面元素
2、提交后的表单处理
界面元素
1、使用 <form> 元素创建表单
2、在 <form> 元素中添加其他表单可以包含的控件元素

# 表单元素 <form>

定义表单：使用 <form></form > 标记
主要属性
action 定义表单被提交时发生的动作，通常包含服务方脚本的 URL
method 指出表单数据 提交的方式，取值为 get 或者 post
enctype 表单数据进行编码的方式
name 表单名称

# 表单控件

表单可以包含很多不同类型的表单控件
表单控件元素是包含在表单元素中具有可视化外观的 HTML 元素，用于访问者输入信息
表单控件元素有
input 元素文本输入控件、按钮、单选、复选、选项卡、文件文件框和隐藏控件
textarea 元素
select 和 option 元素
其他元素

# <input> 元素

# <input> 元素

<input> 元素用于收集用户信息
该元素是一个单标记，语法为:<input />
主要属性
type 根据不同的 type 属性值，可以创建各种类型的输入字段
value 控件的数据
name 控件的名称

# 文本框与密码框

主要属性
value 由访问者自由输入的任何文本
maxlength 限制输入的字符数
readonly 限制文本控件只读

# 单选框和复选框

主要属性
value 文本
name 分组
checked 设置选中

# 按钮

主要属性
value 按钮的标题文本

# 隐藏域和文件选择框

# 其他控件

# <label> 元素

主要属性
for 表示与该元素相联系的控件的 id 值
作用将文本与控件联系在一起后，单击文本，效果就同单击控件一样

# 选项框

<selext> 创建选项框
name 选项框命名
size 大于 1，则为滚动列表
multiple 设置多选
<option> 选项
value 选项的值
selected 预选中

# <textarea> 元素

主要属性
cols 指定列数
rows 指定行数
readonly 只读

# 为控件分组

<fieldset> 分组
<legend> 指定标题
