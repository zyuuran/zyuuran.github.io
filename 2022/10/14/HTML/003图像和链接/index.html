# URL

# url

url(Uniform Resource Locator) 统一资源定位器，用来识别网络中的任何文件

组成
协议 主机名 路径名 文件名

https://ranshao2002.github.io/2022/10/14/HTML/003图像和链接/index.html
https:协议名
ranshao2002.github.io:主机名
/2022/10/14/HTML/003图像和链接/:路径
index.html:文件名

# 相对路径和绝对路径

相对路径以当前文件目录为准

test/index.html
./test/index.html

绝对路径以服务器主机为根目录

https://127.0.0.1:10000/index.html

# 图像元素 <img>

必用属性 src
可选属性 widthheight

<img width="100px" src="./test.jpg" />
表示加载test.jpg这张图片，高度缩放至100像素

# 链接

# 链接元素 <a>

语法
<a href=""target=""> 文本 </a>
href : 链接 URL
target : 目标
