# 表格的作用

表格通常是用来组织结构话的信息
表格是一些被称为单元格的矩形框架按照从左到右，从上到下的顺序排列在一起而形成的
表格的数据保存在单元格里

# 创建表格

定义表格使用成对的 <table></table > 标记
创建表行使用成对的 <tr></tr > 标记
创建单元格使用成对的 <></td> 标记

# 表格的常用属性

<table> 元素 borderwidth/heightaligncellpaddingcellspacing
<tr> 元素 align/valign
<td> 元素 align/valign/width/height/colspan/rowspan

# 表格标题 <caption>

使用 <caption> 元素定义标题
<caption> 标签必须紧随 < table > 标签之后，且只能对每个表格定义一个标题

# 行分组

表格可以分为 3 个部分：表头、表主题和表尾
表头行分组:<thead></thead>
表主题分组:<tbody></tbody>
表尾行分组:<tfoot></tfoot>
