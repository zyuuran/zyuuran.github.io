# XSS 简介

网站脚本攻击（Cross Site Script）缩写 (XSS) 本来缩写为 CSS，但是为了和层叠样式表 Cascading Style Sheet,CSS 有区别，所以叫做 XSS

# 被忽视的角落：Flash XSS

可选值allowScriptAccess 参数
always对于 HTML 的通信也就是执行 JavaScript 不做任何限制
sameDomain只允许来自于笨域的 Flash 与 Html 通信，这是默认值
never绝对禁止 Flash 与页面通信
可选值allowNetworking
all允许使用所有的网络通信，默认值
internalFlash 不能与浏览器通信，但是可以调用其他的 API
none禁止任何网络通信

Cookie 使用过程
Step1浏览器向服务器发起请求，这时候没有 Cookie
Step2服务器返回时发送 Set-Cookie 头，向客户浏览器卸乳 Cookie
Step3在该 Cookie 到期前，浏览器访问该域下的所有页面，都将发送该 Cookie
