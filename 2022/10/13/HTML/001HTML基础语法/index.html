# HTML 概述

# Web 的工作原理

服务器通过 HTTP 协议进行通信，HTTP 协议采用的是请求、响应模式，即客户端发起 HTTP 请求，web 服务器接收并解析处理 HTTP 请求，然后将 HTTP 响应发送给客户端。

# 相关技术

客户端技术：运行于客户端，由浏览器解析运行
HTMLCSSJavaScript 浏览器兼容
服务器端技术：运行于服务端，可以和数据库进行交互
ASP.NETPHPJSP

# 什么是 HTML

HTML(HyperText Markup Language) 超文本标记语言用来设计网页的标记语言
后缀.html.htm

# Web 浏览器

主要功能
代理访问者提交请求
作为 HTML 解释器和内嵌脚本程序执行器
用图形化的方式显示 HTML 文档

# HTML 基础语法

# 标记

使用时必须尖括号括起来
有封闭类型的标记，也有非封闭类型的
封闭类型标记 (双标记)< 标记 > 内容 </ 标记 >
非封闭类型标记 (单标记)< 标记 > 或 < 标记 />


# 元素和属性

元素
即标记
每一对尖括号包围的部分 如：<body></body>

属性
用来修饰元素
每个属性都有值
属性的声明必须位于开始标记里
一个标记的属性不只一个，多个属性之间用空格隔开
多个属性之间不区分先后顺序

元素属性
<body>开始标记
</body>结束标记
<p></p>P 元素
align属性名称
center属性值
<body></body>body 元素

# 通用属性

每个元素都有自己所特有的属性
有些属性是绝大多数元素都支持的属性，称为通用属性
--------
id指定元素的唯一 ID
title规定元素的额外信息
class规定元素的类名
style定义样式

# 注释

<!-- 注释部分 -->
不可嵌套

# HTML 的文档结构

# 文档声明类型

让浏览器清楚文档的版本、类型和风格

# <head> 元素

所有其他元素的容器
紧跟在 <html> 后

# 文档头部内容 <title>

标题元素出现在浏览器顶部
没有属性
必须出现在 <head> 元素中
一个文档只有一个标题元素

# 文档头部内容 <meta>

元数据元素 <meta> 用于定义网页的基本信息
空标记
常用属性：content、http-equiv
